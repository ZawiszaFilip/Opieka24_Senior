Po zainstalowaniu i uruchomieniu aplikacji użytkownik:
Widzi ekran startowy z informacjami a następnie ma możliwość założenia konta lub zalogowania.
Wybiera, czy chce korzystać z aplikacji jako senior, czy jako wolontariusz.
Wypełnia prosty formularz z podstawowymi danymi.
Po przejściu do głównego ekranu może:
- dodać lub przeglądać prośby o pomoc / oferty pomocy,
- ustawiać przypomnienia (np. o lekach, wizytach),
- zarządzać swoim profilem i danymi.
Od strony technicznej projekt jest typowym projektem Android w Kotlinie, zorganizowanym tak, aby można go było bez problemu uruchomić w Android Studio (repozytorium GitHub, integracja z Gradle).
